# Agenda de Contatos em Java

Aplicação desktop desenvolvida em **Java Swing** para gerenciamento de contatos.

## Funcionalidades
- Cadastro de contatos
- Listagem em tabela
- Busca de contatos por texto
- Interface gráfica simples e funcional

## Tecnologias
- Java
- Java Swing
- NetBeans

## Objetivo
Projeto desenvolvido para praticar conceitos de programação em Java,
organização de código e versionamento com Git e GitHub.
